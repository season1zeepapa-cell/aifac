# Brief: 신발 전문 쇼핑몰

> 아디다스 스타일의 신발 전문 쇼핑몰 웹앱 — 회원 전용, 토스페이먼츠 결제

## Requirements
- [x] 1. 회원가입/로그인 페이지 — 아이디/비밀번호 기반, 세션 유지
- [x] 2. 모든 페이지 로그인 필수 — 비로그인 시 로그인 페이지로 리다이렉트
- [x] 3. 메인 페이지 — 신발 상품 10개 목록 표시 (이미지, 이름, 가격)
- [x] 4. 상품 상세 페이지 — 개별 상품 정보, 사이즈 선택, 장바구니 담기
- [x] 5. 장바구니 페이지 — 담은 상품 목록, 수량 변경, 삭제, 총액 표시
- [x] 6. 결제 페이지 — 토스페이먼츠 위젯으로 결제 처리
- [x] 7. server.js — Express 기반 API (상품, 회원, 장바구니, 주문)
- [x] 8. Supabase 연동 — 모든 테이블 `shopping_` prefix 사용
- [x] 9. 상품 데이터 — 신발 10개 (운동화, 러닝화, 슬리퍼 등) seed 데이터 포함

## Constraints
- `workspace/shopping/` 폴더에 구성
- CDN React + Tailwind CSS (빌드 도구 없음)
- Node.js + Express (server.js 단일 파일)
- Supabase 외부 DB (연결 스트링 별도 제공)
- 토스페이먼츠 skill 별도 제공 예정

## Non-goals
- 비회원 브라우징 (게스트 접근 없음)
- 관리자 페이지
- 실시간 재고 관리
- 상품 검색/필터링 고급 기능

## Style
- 아디다스 스타일 — 흑백 기반, 볼드 타이포그래피, 스포티한 레이아웃
- 깔끔하고 대담한 UI

## Key Concepts
- **shopping_ prefix**: 모든 Supabase 테이블명 앞에 붙는 접두어
- **세션 인증**: 로그인 후 서버가 세션을 유지해서 매 요청마다 로그인 상태 확인
- **CDN React**: 빌드 없이 HTML 파일에서 `<script>` 태그로 React 사용
