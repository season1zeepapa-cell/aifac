# Brief: 영상정보관리사 기출 오답 해설 사이트

> Node.js + Express 백엔드 기반, Gemini/OpenAI AI 해설 기능이 있는 오답노트 웹 앱

## Requirements
- [ ] 1. Node.js + Express 서버로 정적 파일 서빙 및 AI API 프록시 제공
- [ ] 2. 110문항 문제 카드 목록 표시 (사이드바 목차 + 카드 리스트)
- [ ] 3. 각 카드에 "Gemini 해설" / "OpenAI 해설" 버튼 2개 제공
- [ ] 4. 버튼 클릭 시 AI 응답을 카드 내에 실시간 표시
- [ ] 5. 기본 요청: HTML 텍스트 추출 전송 / 옵션 토글로 이미지(base64) 전송 전환
- [ ] 6. API 키(.env)는 서버에서만 관리 — 클라이언트에 노출 안 됨
- [ ] 7. 다크모드 토글 (라이트/다크 전환)
- [ ] 8. 모바일 완전 반응형 (사이드바 → 하단 네비)
- [ ] 9. 기존 error.html은 error_backup.html로 백업

## Constraints
- DB 없음 — 모든 데이터는 HTML에 정적으로 포함
- 모델: Gemini 1.5 Pro / GPT-4o (고성능)
- 배포: Vercel

## Non-goals
- 사용자 인증/로그인 없음
- 오답 저장/통계 기능 없음
- 문제 추가/편집 CMS 없음

## Style
- 학습용 — 가독성 최우선 (충분한 줄간격, 큰 폰트, 명확한 대비)
- 다크모드 기본 지원
- 모던 미니멀 (인디고/바이올렛 포인트 컬러)
- 모바일 퍼스트

## Key Concepts
- **오답노트**: 틀린 문제만 모아놓은 복습용 학습 도구
- **AI 해설**: 버튼 1클릭으로 Gemini 또는 OpenAI가 해당 문제의 해설 제공
- **텍스트/이미지 모드**: 기본은 HTML 텍스트로 AI에 전달, 토글 시 문제 이미지(q001.png 등) base64 전송
- **영상정보관리사**: CCTV/영상보안 관련 국가자격증
